Today we interact with computers is generally via a user interface of some sort. Screens are laid out with elements for us to use to interact with the software and in general its pretty easy to get things done until it’s not. When it comes to business applications the learning curve can be steep and often specialist knowledge is required to get the most out of the software. If you consider accounting software, an accountant not only needs to understand all the principals of accounting but also needs a decent set of spreadsheeting skills and probably a couple of accounting packages as well.

By and large the user experience (UX) is well thought out and tries to be intuitive and easy to use with varying degrees of sucess. We humans, you see, are quite varied and we see things differently. What is obvious to me is not to somone else especially of different age groups and demographics. The one thing we’re mostly good at is speaking and understanding each other. If we could only tell the machine what we want then that would be much easier. Enter AI …

The way we interact with machines is about to change quite dramatically. Microsoft announced a suite of copilot tools in their 2023 build conference and in early 2024 started showing beta and prototype versions of their promises. They have baked copilots into the very operating system so now you won’t really need to learn the nuances of software because you can just directly tell the copilot the outcome you want and it will operate the necessary software for you. You won’t need to know how to use a computer anymore. That will become a thing of the past.

What does UX look like in this world? Do we still design for human users or do we optimise for copilots even if it makes it less usable for a human user? If you think about it, we don’t actually need a user interface at all. The copilot is perfectly capable of interacting with web apis directly, can write and execute code on the fly and is built into the operating system. At some point we will only need to interact with the operating system and how it acheives the things we ask of it become abstracted. We tell it to send an email and dictate the contents (or not) and we don’t really know how its sent. That’s not really any different today though … how many people know what happens after they click that send button?

If we consider Apple’s new VisionOS things start to get funky. Here we have a completely immersive computing experience.
